# 秘書 AI 自救班 LP｜Y2K Mood Board × Collage Graffiti

> 把整頁 LP 做成「**她私下會截圖收藏的那種 mood board**」。
> 9:16 直式為主、桌機橫式對應，FB / IG / 小紅書 / 限動分享 主投放規格。
> 用於：FB / IG 冷流量廣告 LP、限動轉貼、KOL 推薦、私群分享。

## 設計總方針

### 美學風格定位

混合三種爆款視覺語言：

1. **真人攝影**（電影感生活照、film grain、自然光）
2. **拼貼風 Collage**（撕貼紙、Polaroid、washi tape、票根、便利貼、咖啡漬、紙質紋理）
3. **塗鴉風 Graffiti / Doodle**（手寫字、Marker pen 圈點、Caveat 體、噴漆、箭頭）

合在一起就是 IG / Pinterest / 小紅書最爆紅的 **「Indie Y2K Mood Board」** 視覺。

### 與「潮牌報紙版」的本質差異

潮牌報紙版是**新聞編輯室**的設計語法（editorial × brutal × streetwear）：
- 一切都「**印出來的**」
- 對齊、欄位、版頭、印章
- 像翻 i-D Magazine / Off-White lookbook

Y2K 拼貼爆款版是**她私人筆記本**的設計語法（mood board × scrapbook × diary）：
- 一切都「**手貼上去的**」
- 撕邊、傾斜、便利貼、polaroid、Marker 圈點
- 像翻 IG 收藏夾 / Pinterest board / 一本日記

| 維度 | 潮牌報紙版 | Y2K 拼貼爆款版 |
|------|-----------|--------------|
| 隱喻 | 報紙 / 雜誌 / lookbook | 日記 / mood board / scrapbook |
| 觸感 | 印刷 / 油墨 / 對齊 | 撕貼 / 膠帶 / 手寫 |
| 字體 | Anton / Playfair / Space Mono | Anton / Permanent Marker / Caveat |
| 色塊 | 大面積平塗（鈷藍轉場） | 多片小色塊 + Polaroid 散落 |
| 互動 | 卡片傾斜、hover 拉正 | 整頁飄 polaroid、貼紙隨機角度 |
| 情緒 | 「**這課很有態度**」 | 「**這課很懂我**」 |

### 三層構圖原則

```
┌────────────────────┐
│  ↑ 塗鴉層           │  ← 手寫字、圈點、Marker 箭頭、噴漆
│  ↑ 拼貼層           │  ← Polaroid、washi tape、票根、便利貼、撕貼紙
│  ↑ 真人攝影層       │  ← 電影感寫實人物與場景
└────────────────────┘
```

由下往上疊：先**真人**、再**拼貼**、再**塗鴉**。每個 ACT 的視覺都遵守這個分層邏輯。

### 統一規格

| 項目 | 規格 |
|---|---|
| LP 容器寬度 | 桌機 max 1280px、手機 100vw |
| 圖片主比例 | 直式 9:16（1080×1920）、橫式 16:9（1920×1080）、方形 1:1（1080×1080） |
| 解析度 | 標準 1080px 短邊，重點圖出 2160px |
| 色彩 | sRGB |
| 出血 | 重要文字距邊 ≥ 5%（避開拼貼遮蓋） |
| 字體 | 標題：Anton + 思源黑 Black<br>內文：Inter / Space Grotesk + 思源黑 Regular<br>手寫塗鴉：Caveat / Permanent Marker / Indie Flower<br>標籤：Space Mono |
| 檔名 | `IMG-Y01_主場景_茶水間愣神_v1.png` |

## 用色系統（7 色 + 紙底 + 多片膠帶）

### Token

```css
--paper: #f5efe2;        /* 米紙底，比潮牌版略黃、像舊筆記本 */
--paper-warm: #ede4cf;
--cream-100: #fffdf3;
--ink: #0d0d0d;          /* 墨黑，手寫感、不刺眼 */
--highlight: #faff00;    /* 螢光黃，蓋章 / 高光（比潮牌黃更刺）*/
--bubblegum: #ff5fa2;    /* 桃粉，sticky note / Y2K 主色 */
--chrome: #c0c8d0;       /* Y2K 銀，metallic accent / 反光 */
--marker-green: #6bd968; /* Marker pen 草綠，圈點 / 簽到 */
--tomato: #ff3838;       /* 番茄紅，警告 / 主 CTA / 紅章 */
--washi-blue: #7bb6d9;   /* 和紙膠帶藍，斜貼裝飾 */
```

### 各色的角色

| 色 | 用途 | 心理任務 |
|----|------|---------|
| 米紙底 | 整本 base | **像她的筆記本** |
| 墨黑 | 主文、手寫字 | **私密、像鋼筆** |
| 螢光黃 | 圈點、highlighter、polaroid 框 | **「這句我要記住」** |
| 桃粉 | sticky note、polaroid、女性元素 | **Y2K 甜美感** |
| Y2K 銀 | chrome 字、反光元素 | **千禧復古** |
| Marker 綠 | 簽到章、check mark、scrapbook tick | **「對對對」的點頭** |
| 番茄紅 | CTA、紅章、限定 | **行動觸發** |
| 和紙藍 | washi tape、polaroid 配色 | **裝飾平衡** |

### 配色比例（黃金法則）

```
紙底 55% + 墨黑 18%
  + (螢光黃 8% + 桃粉 7% + 番茄紅 6% + 銀 3% + 綠 2% + 藍 1%) = 100%
```

- **紙底要夠多**（才像筆記本）
- **彩色點綴均勻**（不偏單色、要 mood board 的雜花感）
- **每屏至少 3 種彩色「貼紙」分佈**（螢光黃 + 桃粉 + 任一色）

### Section 之間的色彩節奏

```
ACT 01 ─ 米底 + 大量 polaroid + 桃粉便利貼 + 螢光黃圈點  （hero 散落感）
ACT 02 ─ 米底 + 1,200 巨型 chrome 字 + 名片拼貼大爆炸     （數字 mood board）
ACT 03 ─ 米底 + 4 個 scrapbook 拼貼塊（每塊不同色票）     （像 4 頁日記）
ACT 04 ─ 奶白底 + 講師 polaroid + 日記手寫雙欄           （沉靜、私密）
ACT 05 ─ 米底 + 黃 polaroid 拼貼 + 紅 CTA 大貼紙          （收口、寄信封）
COLOPHON ─ 墨黑底 + 螢光黃 + 撕貼紙效果                   （壓底）
```

整本滑下來像**翻她私人筆記**，每頁都像新的拼貼 spread。

## 字體系統（5 字型混排）

### Token

```css
'Anton'                /* 超粗壓縮顯示字、大標題、Polaroid 邊字 */
'Permanent Marker'     /* Marker pen 手寫塗鴉、圈點、KEY 字 */
'Caveat'               /* 鋼筆手寫、便利貼、內心獨白 */
'Indie Flower'         /* 圓潤手寫、Polaroid 標題 */
'Space Mono'           /* 等寬，日期 / 編號 / 票根 */
'Inter'                /* 現代無襯線，內文 */
'Noto Sans TC' 900     /* 中文無襯線，大標 */
'Noto Sans TC' 400-500 /* 中文內文 */
```

### 字體角色分工

| 場景 | 字型 | 例子 |
|------|------|------|
| **超大數字 / 大標** | Anton + 思源黑 900 | `1,200` `01` `妳會什麼?` |
| **手寫塗鴉註記** | Permanent Marker | `← THIS ONE` `★ KEY` `WTF` `OMG` |
| **內心獨白便利貼** | Caveat | `「她也愣過？」` `我也想要這個` |
| **Polaroid 邊白標題** | Indie Flower | `MAY 31, 2026` `茶水間` `coffee break` |
| **票根 / 編號 / 標籤** | Space Mono 700 | `ISSUE 07` `NTD 500` `VOL.Y2K` |
| **內文** | Inter + 思源 TC | 段落、卡片描述 |

### 為什麼混 5 種字體

mood board 的本質是**手感混搭**：
- 不同筆、不同貼紙、不同時刻的字，全都不一樣
- 一致性等於「機器印的」、混搭等於「**手做的**」
- 像她真的在筆記本上**東貼一個西貼一個**

讀者解析 hierarchy 不靠字級、靠**字型暗示**：
- 看到 Anton → 這是大標 / 數字
- 看到 Permanent Marker → 這是她**事後加上去的圈點**
- 看到 Caveat → 這是她**寫給自己的話**
- 看到 Space Mono → 這是日期 / 標籤
- 看到 Indie Flower → 這是 Polaroid 下面的小字

## 工作流（建議）

### Option A：GPT-image-1 / DALL-E 3 一次跑（最快）

1. 跑 12 張提示詞（A 主圖 5、B 素材 4、C 背景 3）
2. 每張跑 2-3 個變體挑最好
3. 主圖以「**真人 + 自然光 + film grain**」為主
4. 素材以「**透明背景 PNG / isolated on white**」為主
5. 圖到手後直接嫁進 HTML（拼貼效果用 CSS transform / box-shadow / filter 完成）

### Option B：分層合成（最精準）

1. **GPT-image-1** 跑真人主視覺（5 張，IMG-Y01 至 Y05）
2. **GPT-image-1** 跑拼貼素材（IMG-Y06 至 Y09，透明背景）
3. **Photoshop / Figma** 在主圖上手動疊膠帶、polaroid 邊框、塗鴉
4. 重要塗鴉直接在 CSS 用 Permanent Marker font 寫，**不烤進圖裡**（保留可改性）

### Option C：純 CSS 拼貼（圖片極簡）

1. 只跑 5 張主圖（IMG-Y01 至 Y05）
2. 所有拼貼裝飾全部用 **CSS pseudo-element + transform**
3. 字型 + 框線 + 旋轉 + box-shadow 模擬 polaroid / washi tape / sticky note
4. **零素材依賴**、零 dependencies

**推薦：Option A（先跑全套）+ Option C（CSS 補位）**

主圖必跑、素材跑了用了就用、沒跑用 CSS 撐住。

## 與「真人海報庫 _README.md」的對照

該 README 在 `_AI參考頭像提示詞/` 用於做**5 人合成主視覺海報**。
本 LP 借用同套美學語言，但落地對象是「**1 頁式 LP**」。

| 維度 | 角色海報庫 | 本 LP（Y7） |
|------|----------|------------|
| 產出物 | 5 張 9:16 海報 + 1 張 5 人合成 | 1 頁完整 LP（5 屏） |
| 主用途 | IG / 抖音 / 小紅書 貼文 | FB / IG 廣告 LP / 分享連結 |
| 真人元素 | 5 個角色 token 一致性 | 玉婷 + Lisa 兩個 token 一致性 |
| 拼貼密度 | 每張 100%（海報就是拼貼） | 每屏 60-80%（要保留閱讀區） |
| 塗鴉密度 | 高 | 中（避免 LP 太亂、跳出率高） |
| 字體 | 同套 | 同套 + Inter / 思源 TC（內文要好讀） |

**承接邏輯**：5 張海報跑出來後，**可從海報截角色頭像直接嫁進 LP**（保 token 一致性）。

## 與 Codex 對接（給 AI 生圖）

提示詞檔案：**`圖片生成指令.md`**

每張圖含：
- `[ID]` 圖片編號（IMG-Y01...）
- `[USE]` 用途（哪一屏、哪個位置）
- `[SPEC]` 規格（比例、解析度、檔名）
- `[PROMPT]` 主提示詞（GPT-image-1 自然語言）
- `[NEGATIVE]` 不要的元素
- `[VARIANTS]` 變體建議（同人物不同表情、同場景不同角度）

**Codex 一鍵跑全套**指令也在文末。
